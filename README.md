# مطعم رستورانت - موقع قائمة طعام

موقع قائمة طعام لمطعم بتصميم عربي فخم، متجاوب، يعتمد على البيانات (JSON) ولوحة تحكم (Decap CMS).

## هيكل المشروع

```
Restorant/
├── index.html          ← الموقع العام (RTL عربي، متجاوب)
├── styles.css          ← التصميم (مظهر فاخر، حركات انتقال)
├── app.js              ← تحميل البيانات من JSON، البحث، تصفية الفئات
├── data/
│   └── menu.json       ← بيانات القائمة (الفئات والأصناف)
├── admin/
│   ├── index.html      ← لوحة تحكم Decap CMS
│   └── config.yml      ← إعدادات لوحة التحكم
├── assets/
│   └── images/         ← صور الأطباق
├── _redirects          ← توجيه Netlify
└── netlify.toml        ← إعدادات بناء Netlify
```

## كيف تعمل لوحة التحكم (/admin)

### الوصول للوحة التحكم
1. انشر الموقع على Netlify (انظر تعليمات النشر أدناه)
2. اذهب إلى `https://your-site.netlify.app/admin/`
3. سجّل الدخول عبر Netlify Identity

### إدارة القائمة
بعد الدخول، ستجد:
- **الفئات**: قائمة بجميع فئات الأطعمة (المقبلات، الأطباق الرئيسية، إلخ)
  - إضافة فئة جديدة: اضغط "أضف" في حقل الفئات
  - حذف فئة: اضغط زر الحذف بجانب الفئة
- **عناصر القائمة**: قائمة بجميع الأطباق
  - إضافة صنف جديد: اضغط "أضف" في حقل العناصر
  - تعديل صنف: اضغط على الصنف لتعديل (الاسم، السعر، الوصف، الصورة، الوسوم)
  - حذف صنف: اضغط زر الحذف
  - إخفاء صنف: اجعل خيار "متاح" = لا

### الوسوم المتاحة
أضف وسوم للأصناف مثل:
- `popular` / `شائع` ← يظهر شارة "شائع"
- `best-seller` ← يظهر شارة "الأكثر مبيعاً"
- `new` ← يظهر شارة "جديد"
- `spicy` ← يظهر شارة "حار"
- `chef-special` ← يظهر شارة "خاص"
- `vegetarian`, `vegan`, `gluten-free` ← تظهر كوسوم عادية

## نشر الموقع على Netlify

### خطوات سريعة:
1. ارفع المشروع إلى GitHub/GitLab/Bitbucket
2. ادخل إلى [Netlify](https://netlify.com) واضغط "Add new site"
3. اربط مستودعك ( repository)
4. إعدادات البناء:
   - **Build command**: (اتركه فارغاً - موقع ثابت)
   - **Publish directory**: `.` (الجذر)
5. اضغط "Deploy site"

### تفعيل لوحة التحكم (Decap CMS):
1. في لوحة تحكم Netlify، اذهب إلى **Site settings → Identity**
2. اضغط "Enable Identity"
3. اذهب إلى **Settings → Identity → Services → Git Gateway** واضغط "Enable Git Gateway"
4. أضف مستخدمين من خلال **Identity → Invite users**

### اختبار لوحة التحكم محلياً:
```bash
npx netlify-cms-proxy-server
# في نافذة أخرى
npx serve .
```
ثم اذهب إلى `http://localhost:3000/admin/`
